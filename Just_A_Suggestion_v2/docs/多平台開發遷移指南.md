# 🚀 《只是一個建議》多平台開發遷移指南 (V22.5)

如果您需要在另一台電腦上繼續開發此專案，請遵循本指南進行環境恢復與同步。

## 1. 取得最新代碼
在目標電腦上確保已安裝 Git，並執行：
```bash
git pull origin main
```
*這會取得包含 V22.5 視覺引擎（色彩殺手、情緒切換、分鏡解放）在內的所有程式碼。*

## 2. ⚠️ 關鍵環境變數設定 (手動備份)
由於安全性因素，以下文件**未上傳**至 Git，請手動將它們複製到新電腦的對應位置：

### A. `.env` 文件
**放置位置**：`Just_A_Suggestion_v2/.env`
**內容範例**：
```env
GEMINI_API_KEY=您的API金鑰
GCP_PROJECT_ID=您的專案ID
GCP_LOCATION=us-central1
```

### B. `gcp_service_account.json` (GCP 服務帳戶密鑰)
**放置位置**：`config/secrets/gcp_service_account.json`
**重要性**：這是連接 Vertex AI (Imagen 3.0 生圖) 的身分憑證。如果沒有這份檔案，生圖功能將無法運作。

---

## 3. 啟動開發環境
本專案完全基於 Docker 運作，請確保新電腦已安裝 Docker Desktop 並啟動。

**執行命令**：
```bash
docker-compose up -d
```
*這會啟動所有的容器服務，並自動掛載專案目錄。*

## 4. 驗證連線
伺服器啟動後，請訪問：
*   **遊戲網址**：`http://localhost:8002`
*   **後端日誌檢查**：`docker logs -f ai-prison-workspace`

---

## 5. 目前版本狀態紀錄 (V22.5)
- **視覺引擎**：已實裝「色彩殺手協議」，徹底消滅青色滲透。
- **情緒視覺**：恐懼值 > 0.6 會自動切換為漫畫網點風格；低於時為墨染風格。
- **分鏡系統**：已解鎖 Cinematic Framing，AI 會自動選擇鏡頭角度。

---
**Developer 備忘錄**：如果您在新環境遇到 Docker 權限問題，請嘗試執行 `chmod +x start_game.sh` 或聯繫您的 AI 助手 Antigravity。
