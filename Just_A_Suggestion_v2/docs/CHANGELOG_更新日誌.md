# 📝 CHANGELOG (更新日誌)

## [V22.5] - 2026-05-06
### 新增 (Added)
- **情緒風格切換 (Emotional Shifting)**: 於 `build_image_prompt` 加入恐懼值閾值 (0.6) 判定。低於時使用墨染風格，高於時切換為復古漫畫網點風格。
- **不露臉協議 (No-Face Protocol)**: 強制構圖鎖定為背影 (Back view) 或側面 (Side view)，面部永遠隱藏於陰影中。
- **色彩協議整合**: 將 V22.0 的物理詞彙轉譯矩陣與新風格完全整合。

## [V20.1] - 2026-05-05
### 新增 (Added)
- **藝術筆觸模式 (Artistic Brush Mode)**: 全面捨棄攝影寫實風格，轉向高對比度的漫畫排線 (Cross-hatching) 與粗獷鉛筆素描。
- **HAMP 藝術補強**: 對於燈光與反射等詞彙，強制加入「無著色器、粗糙排線」的藝術限制。

## [V19.5] - 2026-05-05
### 新增 (Added)
- **麵包屑解謎系統 (Breadcrumb Puzzles)**: 重構 10 分鐘流程的動機，加入「寫有電話號碼的紙條」與「語音提示」，確保玩家不會卡關。
- **強制怪誕敘事 (Guaranteed Anomalies)**: 設定 100% 機率在每回合生成一個不重複的超現實都市現象。
- **系統架構文件**: 於桌面建立詳細的軟硬體、架構與工作流文件。
