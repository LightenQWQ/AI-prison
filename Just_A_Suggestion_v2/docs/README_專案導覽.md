# 🌧️ AI Prison & Just A Suggestion (只是一個建議)

這是一個以**都市孤寂**與**心理驚悚**為主題的互動實驗專案。
專案最初名為「AI Prison」，探討 AI 在虛擬世界中的行為與牢籠；隨後進化並衍生出目前的核心遊戲：**《只是一個建議 (Just A Suggestion)》**。

---

## 🎮 關於《只是一個建議》V22.5
這是一款基於 Google Gemini 1.5 Flash 與 Vertex AI (Imagen 3.0) 構建的即時動態視覺小說。
玩家將在一個永遠停在凌晨三點、不斷下著冷雨的黑白城市中，透過給予一位迷失青年「建議」，來引導他解開三環謎題（電話亭 ➝ 販賣機 ➝ 地鐵站）。

### 核心特色
1. **雙軌敘事引擎**：分離「對白」與「旁白」，並保證 100% 觸發不可預期的怪誕異常現象。
2. **心理感應視覺系統**：畫面風格會根據主角情緒動態切換。低恐懼時呈現「深邃墨染 (Noir Ink)」，高恐懼時轉向「高對比漫畫網點 (Manga Screen-tone)」。
3. **絕對神祕感鎖定**：強制執行「不露臉協議」，所有鏡頭皆鎖定為背影、側面或深邃剪影，強化都市孤寂感。

## [V22.5] - 2026-05-06
### 新增 (Added)
- **情緒風格切換 (Emotional Shifting)**: 於 `build_image_prompt` 加入恐懼值閾值 (0.6) 判定。低於時使用墨染風格，高於時切換為復古漫畫網點風格。
- **不露臉協議 (No-Face Protocol)**: 強制構圖鎖定為背影 (Back view) 或側面 (Side view)，面部永遠隱藏於陰影中。
- **色彩協議整合**: 將 V22.0 的物理詞彙轉譯矩陣與新風格完全整合。

## [V22.0] - 2026-05-06

## ⚙️ 快速啟動
1. 確保已安裝 Docker 與 Docker Compose。
2. 放入 `.env` 與 `gcp_service_account.json`。
3. 執行 `docker-compose up -d` 啟動伺服器。
4. 打開瀏覽器訪問 `http://localhost:8002` 即可遊玩。
