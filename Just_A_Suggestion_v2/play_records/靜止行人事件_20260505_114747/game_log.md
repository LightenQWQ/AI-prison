# 遊戲遊玩紀錄 - 靜止行人事件

**結局類型**：

### 第 1 回合
**玩家建議**：看看口袋裡的紙條，去最近的公用電話亭。

**角色對話**：紙條？我口袋裡什麼都沒有。為什麼要去電話亭？這裡……這裡很好。雨聲很安靜，我不想去任何地方。

**環境旁白**：黑白色的雨水永不停歇，沖刷著空無一人的街道。你注意到有些雨滴似乎違反了常理，它們在半空中短暫懸停，彷彿猶豫不決，然後才無聲地融入濕透的柏油路面。這座城市的一切都顯得遲滯而沉重。

### 第 2 回合
**玩家建議**：在電話亭撥打紙條上的號碼。

**角色對話**：為什麼要打電話？這裡沒有人……也沒有人會接。這東西早就壞了。別管它了，我們離開這裡好不好？雨好像越來越大了。

**環境旁白**：他緊緊握著冰冷的話筒，手指卻僵在撥盤上，像一隻受驚的小動物。你注意到，電話亭外的雨滴，有幾顆在半空中凝結了整整一秒，才重新墜落，彷彿時間本身也在此地猶豫不決。這座城市的異常，正以微小的方式侵蝕著你和他之間的稀薄連結。

![場景]( turn_2.png )

### 第 3 回合
**玩家建議**：聽聽看電話那頭的聲音，是不是有呼吸聲？

**角色對話**：那不是呼吸聲！是……是風聲，還有雨滴進來的聲音……你不要再說了！

**環境旁白**：男孩猛地將話筒從耳邊推開，彷彿被燙到一般。雨點砸在電話亭的玻璃上，匯集成扭曲的細流。他腳下的積水倒映著褪色的霓虹招牌，卻唯獨沒有映出他的影子。

![場景]( turn_3.png )

### 第 4 回合
**玩家建議**：雖然周遭環境開始變得很奇怪，但請保持冷靜，繼續尋找車站入口。

**角色對話**：你看！他們都停下來了…是不是…是不是我們也可以停下來？我不想再往前走了…車站裡面…有我不想看到的東西。

**環境旁白**：周遭的異常開始加劇。懸浮在半空的雨滴，像無數顆凝固的玻璃珠。前方車站入口處，湧現出一群靜止的、沒有五官的行人，他們維持著行走奔跑的姿態，卻像蠟像般紋絲不動，徹底堵死了去路。時間，彷彿在這裡被抽成了真空。
