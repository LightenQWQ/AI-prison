# 🧠 AI Memory OS 學生/開發者使用指南

歡迎使用 **AI Memory OS 專案工具包**！這是一套專門為 AI (如 Cursor, GitHub Copilot) 與人類協作設計的微型作業系統架構。

## 🚀 第一步：如何開始？
1. **啟動環境**：將解壓縮後的資料夾匯入你的編輯器 (例如 Cursor)。
2. **啟動 AI**：打開編輯器的 AI 對話框 (Chat)，請直接輸入：「**請閱讀 SETUP.md 並啟動環境，接著遵守 AGENTS.md 的規則進行自我介紹。**」
3. **Docker 容器**：所有的程式執行、套件安裝都會被限制在安全的 Docker 容器中，不會弄髒你自己的電腦。
